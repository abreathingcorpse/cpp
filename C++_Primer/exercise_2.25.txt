Determine the types and values of each of the following variables.

a) int* ip, i, &r = i;
b) int i, *ip = 0;
c) int* ip, ip2;

Answer:
a) ip is a pointer to an int.
   i is an integer.
   r is a reference to i.

b) i is an integer.
   ip is a nullptr.

c) ip is an pointer to an int.
   ip2 is an integer.
